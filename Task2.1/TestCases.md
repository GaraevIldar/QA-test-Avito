# Тест - кейсы

## 1) Получить объявление по идентификатору

 - **Эндпоинт:** `{{baseUrl}}/api/1/item/:id`
 - **Метод:** `GET`
 - **Заголовки:** `Accept: application/json`
---
 - При помощи метода `POST {{baseUrl}}/api/1/item` создать объявление
 - Из тела ответа взять значение поля id
 - Использовать значение в path-параметре
---
### OK
**Статус-код 200**

**Схема ответа**
```
[
  {
    "id": "<string>",
    "sellerId": "<integer>",
    "name": "<string>",
    "price": "<integer>",
    "statistics": {
      "likes": "<integer>",
      "viewCount": "<integer>",
      "contacts": "<integer>"
    },
    "createdAt": "<string>"
  },
  {
    "id": "<string>",
    "sellerId": "<integer>",
    "name": "<string>",
    "price": "<integer>",
    "statistics": {
      "likes": "<integer>",
      "viewCount": "<integer>",
      "contacts": "<integer>"
    },
    "createdAt": "<string>"
  }
]										
```
*Схема 1*

---

### Bad Request
**Статус-код 400**

**Схема ответа**
```
{
  "result": {
    "messages": {
      "culpa_b92": "<string>",
      "enim_24f": "<string>",
      "mollit_aa": "<string>"
    },
    "message": "<string>"
  },
  "status": "<string>"
}									
```
*Схема 2*

---

### Not Found
**Статус-код 404**

**Схема ответа**
```
{
  "result": "laborum",
  "status": "cillum enim eiusmod"
}									
```
*Схема 3*

## 2) Сохранить объявление

 - **Эндпоинт:** `{{baseUrl}}/api/1/item`
 - **Метод:** `POST`
 - **Заголовки:**:
   - `Content-Type: application/json`
   - `Accept: application/json`
 
---

### Запрос
**Схема тела**
```
{
  "sellerID": "<integer>",
  "name": "<string>",
  "price": "<integer>",
  "statistics": {

    "viewCount": "<integer>",
    "contacts": "<integer>"
  }
}									
```
