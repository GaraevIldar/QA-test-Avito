## Critical (Критический приоритет)

### ) `GET /api/1/item/:id` возвращает массив `[]` вместо одного объекта `{}`

| Поле                         | Значение                                                                                                                                                                                                      |
|:-----------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Шаги для воспроизведения** | 1. Создать объявление через `POST /api/1/item` и получить его `id`.<br>2. Отправить `GET` запрос на эндпоинт `/api/1/item/{id}` с полученным `{id}`                                                           |
| **Фактический результат**    | Сервер возвращает статус `200 OK` и тело ответа в виде массива, содержащего один объект: `[ { "id": "...", ... } ]`.                                                                                          |
| **Ожидаемый результат**      | Сервер должен возвращать статус `200 OK` и тело ответа в виде **одного объекта**: `{ "id": "...", ... }`.                                                                                                     |
