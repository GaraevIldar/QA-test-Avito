## Task2.1

### Проект с автотестами для API Avito
### Структура
- `tests` - находятся тесты
- `test_data` - находятся данные для тестов
- `utilities` - вспомагательные классы
- `TestCases.md` — полное описание тест-кейсов
- `README.md` — данный файл
- `BUGS.md` — полное описание найденных багов
- `requirements.txt` - файл с зависимостями
- `pytest.ini` - файл  конфигурации Pytest

## Инструкция по запуску тестов

 - Склонировать репозиторий
 - Перейти в каталог с Task2.1
 - Установить зависимости
   ```
   pip install -r requirements.txt
   ```
 - В командной строке выполнить команду для запуска тестов
   ```
   pytest -v
   ```
